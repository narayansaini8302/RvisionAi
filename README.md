# RvisionAI

AI-powered interview preparation platform that analyzes resumes and job descriptions to generate personalized interview strategies.

## 🚀 Features
- 📄 Resume parsing and analysis
- 🤖 AI-powered interview report generation
- 📊 Match scoring and skill gap analysis
- 📝 Customized interview questions
- 📑 PDF export functionality

## 🛠️ Tech Stack
- **Frontend:** React, Vite, TailwindCSS, Framer Motion
- **Backend:** Node.js, Express, MongoDB, Gemini AI
- **Deployment:** Vercel (Frontend), Render (Backend)

## 📦 Setup
1. Clone the repository
2. Install dependencies: 
pm install in both frontend and backend
3. Set up environment variables
4. Run development servers
